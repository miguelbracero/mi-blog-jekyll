HTML y CSS son las tecnologías básicas para construir páginas web. HTML permite estructurar el contenido, mientras que CSS se utiliza para mejorar el aspecto visual.

Estas herramientas son fundamentales porque permiten crear sitios sencillos, navegables y con un diseño atractivo, que es precisamente uno de los objetivos de esta actividad.
